# Design Parameters — BOPM System

## Theme: Dark Militar/Policial
- Background: #0d1117 (muito escuro, quase preto)
- Surface/Card: #161b22 (cinza escuro)
- Border: #30363d (cinza metálico sutil)
- Primary accent: #4a7c59 (verde militar)
- Primary hover: #3d6b4a
- Text primary: #e6edf3 (branco acinzentado)
- Text secondary: #8b949e (cinza claro)
- Danger/alert: #da3633 (vermelho alerta)
- Warning: #d29922 (amarelo âmbar)
- Success: #3fb950 (verde sucesso)

## Typography
- Font: Inter (sans-serif, profissional)
- Headings: font-bold, tracking-wide, uppercase
- Report text: font-mono para o BOPM gerado

## UI Details
- Border radius: 6px (sharp, militar)
- Buttons: uppercase, letter-spacing, sem arredondamento excessivo
- Cards: border com glow sutil no hover
- Inputs: background escuro, borda cinza, focus verde militar
